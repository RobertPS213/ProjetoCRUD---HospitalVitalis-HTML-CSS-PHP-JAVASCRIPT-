**<h2>Hospital Vitalis - Sistema de Gestão Hospitalar</h2>**
Este projeto foi desenvolvido como um sistema de gerenciamento interno para hospitais, focado em operações essenciais de cadastro e controle de dados (CRUD). O objetivo principal foi criar uma interface funcional integrada a um banco de dados para facilitar o fluxo de informações hospitalares.
**<h2>O que o sistema faz?</h2>**
**O Hospital Vitalis permite o controle completo de três pilares principais:**

**Médicos:** Cadastro e listagem de profissionais por especialidade.

**Pacientes:** Registro de prontuários e dados de contato.

**Consultas:** Agendamento e histórico de atendimentos.

**<h2>Tecnologias Utilizadas</h2>**
**Como um projeto full-stack focado na web clássica, utilizei:**

**PHP:** Lógica de backend e conexão com o banco de dados.

**MySQL:** Armazenamento relacional das informações.

**JavaScript:** Validações de formulários e interatividade.

**HTML5 & CSS3:** Estruturação e estilização da interface.

**<h2>Estrutura do Projeto</h2>**
**Para manter o código organizado, o projeto segue uma divisão clara:**

**index.php:** Porta de entrada do sistema.

**config.php:** Configurações de conexão com o Banco de Dados.

**salvar-....php:** Scripts responsáveis pelo processamento das ações (Create, Update, Delete).

**listar-....php:** Visualização dos dados cadastrados.

**<h2>Como rodar o projeto localmente</h2>**

1 - Certifique-se de ter um servidor local instalado (XAMPP, Wamp ou Laragon).

2 - Clone o repositório dentro da pasta htdocs ou equivalente.

3 - Importe o banco de dados SQL (caso disponível no repositório) no seu phpMyAdmin.

4 - Acesse localhost/ProjetoCRUD---HospitalVitalis-HTML-CSS-PHP-JAVASCRIPT- no seu navegador.
